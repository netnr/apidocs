Mobile（移动）

使用$.mobile.defaults重写默认值对象。（该组件自1.4.2版开始可用）

 

属性

属性名 属性值类型 描述 默认值
animation string 导航面板的动画效果类型。 slide
direction string 导航面板的动画效果方向。 left

 

方法

方法名 方法参数 描述
$.mobile.init none 初始化移动面板。
$.mobile.nav from,to,animation,direction

从一个面板导航到另外一个面板。

代码示例：

$.mobile.nav('#p1','#p2','slide','left');

 

$.mobile.go panel,animation,direction

导航到指定面板。

代码示例：

$.mobile.go('#p2');
$.mobile.go('#p3','slide','right');
$.mobile.back none 导航回到上一个面板。
